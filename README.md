# Site Profissional de Streamer

Este é um site totalmente profissional e responsivo criado para streamers de TikTok e YouTube focados em conteúdo de Roblox. O design é moderno, clean e tecnológico, transmitindo credibilidade e profissionalismo.

## Estrutura do Projeto

```
site-profissional/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos CSS
├── js/
│   └── script.js       # JavaScript para interações
└── assets/             # Pasta para imagens e recursos
```

## Funcionalidades

- **Design Responsivo**: Otimizado para desktop e mobile
- **Navegação Suave**: Menu fixo com scroll suave
- **Animações**: Transições suaves e efeitos hover
- **SEO Básico**: Meta tags para motores de busca (incluindo "roblox, scripts")
- **Formulário de Contato**: Para parcerias e propostas
- **Links para Redes Sociais**: TikTok, YouTube, Discord, Twitch

## Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Para desenvolvimento, use uma extensão como "Live Server" no VS Code
3. Personalize o conteúdo:
   - Substitua "SeuNome" pelo seu nickname
   - Atualize os links das redes sociais
   - Adicione sua foto de perfil em `assets/`
   - Modifique a agenda de lives
   - Atualize o e-mail no formulário

## Personalização

### Cores
As cores principais são definidas em `css/styles.css`:
- Preto: #000
- Roxo: #6a0dad
- Azul: #007bff
- Branco: #fff
- Neon: #00ff88

### Tipografia
Usa a fonte Montserrat do Google Fonts.

### Imagens
- Substitua a imagem de placeholder por sua foto real
- Adicione logos ou banners personalizados

## Tecnologias Utilizadas

- HTML5
- CSS3 (Grid, Flexbox, Animações)
- JavaScript (ES6+)
- Font Awesome (ícones)
- Google Fonts
